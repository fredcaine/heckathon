## Concept

The main idea of the game was that it was a command-line interface that supported graphics. This has been done before, but we thought it would be especially clever, and likely different from what most will do.  

We also innovated by making "scene" a command, similar to opening your eyes, as opposed to the more common format of keeping everything on screen at all times.

---

## Features

The game features:
- multiple player classes,  
- special attacks,  
- different rooms that are randomly generated (all featuring special fights and scenarios),  
- saving mechanisms to avoid unfair loss of items,  
- many well drawn sprites,  
- and several other features.
